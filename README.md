## Development

### Local Setup

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000`

### Docker

```bash
docker build -t essanpupil-portfolio .
docker run --rm -v "$PWD":/home/app --network host essanpupil-portfolio
```

### Customization

Edit configuration and content in:
- `_config.yml` — Site settings, resume info, social links
- `_data/` — experience.yml, education.yml, projects.yml, skills.yml, links.yml
- `_layouts/resume.html` — Layout structure

### Deployment

Automatically deployed to GitHub Pages on push to `master`.
